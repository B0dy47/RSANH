What is the Course?

A comprehensive 2-month program designed to transform beginners into proficient back-end developers. Through hands-on projects, you'll master PHP, MySQL, authentication systems, and modern server-side development practices while building secure web applications.

Course Plan Overview

  • 24 sessions (3 hours each) over 12 weeks
  • Hands-on projects and real-world exercises
  • Final capstone project presentation
  • In-depth coverage of PHP, MySQL, API development, and security

What Will You Learn?

  • PHP Programming Fundamentals and Advanced Techniques
  • MySQL Database Design and Management
  • Building CRUD Applications and User Authentication Systems
  • REST API Development and MVC Architecture
  • Advanced Security, Debugging, and Performance Optimization

What Will the Course Prepare You For?

  • Building secure web applications
  • Developing RESTful APIs and MVC-based projects
  • Managing databases and implementing robust authentication systems
  • Pursuing back-end developer roles

Prerequisites

  • Basic understanding of programming
  • Completion of the Front-End Course
  • Logical problem-solving skills

Course Curriculum

Total Sessions: 24

Session Duration: 3 Hours

Sessions per Week: 2

Total Duration: 3 Months

Week 1

  • Session 1: Introduction to PHP & Basic Syntax
    • Overview of PHP and Back-End Development
    • PHP Basics: Variables, Data Types, and Operators
    • Control Structures: if, else, switch
    • Loops (for, while, foreach)
  • Session 2: Functions, Arrays & Error Handling
    • Defining and Using Functions
    • Working with Arrays
    • Basic Error Handling (try/catch, error reporting)

Week 2

  • Session 3: PHP Forms & Superglobals
    • Building and Understanding HTML Forms
    • Handling Form Data with $_GET and $_POST
    • Overview of Superglobals: $_SESSION and $_COOKIE
    • Form Validation & Security Practices
  • Session 4: Advanced Form Handling & Data Sanitization
    • In-depth Input Validation Techniques
    • Data Sanitization and Security
    • Hands-on Exercise: Develop a robust contact form

Week 3

  • Session 5: Introduction to MySQL & Database Fundamentals
    • Why Use Databases?
    • Database Design Essentials
    • Overview of SQL Commands (SELECT, INSERT, UPDATE, DELETE)
  • Session 6: Connecting PHP to MySQL & Basic Queries
    • Setting Up MySQL and Establishing Connections
    • Executing Basic Queries with PHP
    • Hands-on Exercise: Fetch and display data from a database

Week 4

  • Session 7: CRUD Operations with PHP & MySQL
    • Performing Create, Read, Update, Delete Operations
    • Dynamic Data Display on Webpages
    • Hands-on Exercise: Build a Simple CRUD Application
  • Session 8: User Authentication – Part 1
    • Introduction to User Authentication
    • Managing Sessions for User Data
    • Building a User Registration System
    • Secure Password Storage (password_hash, password_verify)

Week 5

  • Session 9: User Authentication – Part 2
    • Creating a Login System with Session Management
    • Advanced Security Practices for User Authentication
    • Hands-on Exercise: Complete your user authentication module
  • Session 10: Handling File Uploads & File Management
    • Validating File Types and Sizes
    • Organizing and Storing Uploaded Files
    • Error Handling in File Uploads
    • Hands-on Exercise: Implement a file upload feature

Week 6

  • Session 11: Advanced PHP Techniques and Security
    • Prepared Statements to Prevent SQL Injection
    • Implementing Pagination for Large Datasets
    • Hands-on Exercise: Secure Database Operations
  • Session 12: Advanced Error Handling & Debugging
    • Deep Dive into Exception Handling (try/catch)
    • Debugging Techniques and Tools
    • Logging, Exception Management, and Best Practices

Week 7

  • Session 13: Advanced MySQL – Joins & Relationships
    • Using JOINs to Combine Data from Multiple Tables
    • Understanding One-to-Many & Many-to-Many Relationships
    • Hands-on Exercise: Build a blog system with posts and comments
  • Session 14: Query Optimization & Database Performance
    • Optimizing SQL Queries and Indexing
    • Performance Tuning Techniques
    • Best Practices for Efficient Database Management

Week 8

  • Session 15: Introduction to APIs & REST Principles
    • Overview of APIs in Back-End Development
    • Consuming External APIs with PHP (cURL, file_get_contents())
    • Hands-on Exercise: Fetch data from an API
  • Session 16: Building Your Own API
    • Designing RESTful APIs
    • Implementing Routing and JSON Responses in PHP
    • Hands-on Exercise: Create a basic API service

Week 9

  • Session 17: Introduction to MVC Architecture
    • Understanding MVC: Model, View, Controller
    • Benefits of Using MVC in PHP Applications
    • Hands-on Exercise: Structure a simple MVC project
  • Session 18: Working with PHP Frameworks
    • Overview of Popular PHP Frameworks (e.g., Laravel, CodeIgniter)
    • Setting Up a Basic Project Using MVC Principles
    • Hands-on Exercise: Build a module using a PHP framework

Week 10

  • Session 19: Debugging and Performance Optimization
    • Advanced Debugging Techniques and Profiling Tools
    • Implementing Caching Strategies in PHP
  • Session 20: Security Best Practices for Back-End Development
    • Identifying Common Vulnerabilities (XSS, CSRF, SQL Injection)
    • Implementing Comprehensive Security Measures
    • Hands-on Exercise: Secure an existing PHP application

Week 11

  • Session 21: Final Project Development – Part 1
    • Project Planning and Design
    • Integrating Multiple Back-End Components
    • Initial Development and Setup
  • Session 22: Final Project Development – Part 2
    • Continued Project Development and Feature Integration
    • Debugging and Incorporating Feedback
    • Preparing for the Final Presentation

Week 12

  • Session 23: Final Project Finalization
    • Finalizing Features and Code Review
    • Comprehensive Testing and Debugging
    • Rehearsing the Final Presentation
  • Session 24: Final Project Presentation & Course Wrap-Up
    • Presentation of Final Projects
    • Course Review, Q&A, and Discussion of Next Steps
